# Storage State Artifacts

This directory contains Playwright authentication storageState files generated by setup scripts.

## Files

- `member.json` - Generated by `member.setup.ts`, contains mock member auth state
- `*.json` - Other storageState files (gitignored)

## Usage

These files are automatically generated when running Playwright tests with the setup project:

```bash
npx playwright test --project=setup
```

The setup scripts run before the main test suites and create storage state files that fixtures can load.

## Phase 1 Implementation

Per `docs/plan/IS-62/plan-00.md` Phase 1:
- `memberSession` fixture loads `member.json` to provide authenticated member state
- `adminSession` fixture composes `memberSession` and adds admin-specific storage

## Note

These files contain mock/test data only and do not include real credentials.
