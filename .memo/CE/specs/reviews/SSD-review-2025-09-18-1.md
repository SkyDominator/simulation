## 2. Critical Gaps and Inconsistencies

### 2.1 Missing Static Privacy Policy File

**Issue**: The SSD mentions static file fallback for privacy policy, but the file `docs/privacy-policy-ko.md` does not exist

**Evidence**: 
- Backend routes.py line 467 attempts to read this file
- File search returns no results
- This would cause FileNotFoundError in production

**Impact**: Privacy policy endpoint will fail when database is unavailable

**SSD Section**: Section 7 - Security & Authentication, Section 9 - API Contracts

### 2.2 Incomplete API Contract Descriptions

**Location**: Section 9 - API Contracts

**Issues**:

1. **POST /api/consents contract incomplete**:
   ```javascript
   // Current in SSD (incomplete):
   {
     req: { user_hash: string, consent_type: string, consent_version: string, 
   
   // Should be (from implementation):
   {
     req: { user_hash: string, consent_type: string, consent_version: string, 
            ip_address?: string, user_agent?: string },
     res: { user_hash: string, consent_type: string, consent_version: string, 
            consent_given_at: string, ip_address?: string, user_agent?: string }
   }
   ```

2. **POST /api/simulation/create contract incomplete**:
   ```javascript
   // Current in SSD (incomplete):
   {
     req: { plan_id: string, starting_company_round: number, current_company_round: number, 
   
   // Should be (from implementation):
   {
     req: { plan_id: string, starting_company_round: number, current_company_round: number, 
            simulation_rounds: number, scheduled_payment: Record<string, number>,
            sales_achievement_rates?: Record<string, number> },
     res: { simulation_id: string, plan_id: string, message: string, success: boolean }
   }
   ```

3. **GET /api/privacy-policy response incorrect**:
   ```javascript
   // Current in SSD:
   {
     res: { version: string, last_updated: string, content: string, 
            success: boolean, source: "db" }
   }
   
   // Should be (from implementation):
   {
     res: { version: string, last_updated: string, content: string, 
            success: boolean, source: "db" | "static-file", locale?: string }
   }
   ```
